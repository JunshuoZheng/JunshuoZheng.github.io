优化文章编辑体验

使用Obsidian管理文章

https://obsidian.md/
Obsidian是一个基于Markdown文件的本地知识管理软件。除了其特色的“双向链接”功能，其基础的Markdown编辑功能做的也十分合格。

用来管理博客文章，作为Markdown编辑器，可以关闭大部分特色功能。Obsidian的软件设置与仓库关联，修改设置不会影响其他仓库。设置保存在仓库.obsidian文件夹下，方便备份和转移。

安装插件：

  • Obsidian Git：无需打开命令行，直接在Obsidian中提交.
  • Image Auto Upload Plugin：支持Picgo的插件.

使用PicGo自动上传图片至图床.

https://github.com/Molunerfinn/PicGo
PicGo 支持图床多，插件丰富，使用非常方便。配合Obsidian的插件可以直接将图片拖入文章中。
